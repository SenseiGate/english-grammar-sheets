CEFR: A1

Lesson 8: There is / There are

📚 1. Grammar Explanation（文法の説明）

We use There is / There are to say something exists or is present.

物や人の存在を表すときに使います。
There is / There are = あります（物）/ います（人・動物）

There is = one thing. There are = two or more things.

There is は単数、There are は複数に使います。

FormUseExample
There isSingular nounThere is a cat.
There arePlural nounThere are two cats.

Negative: There is not / There are not

否定文は「There is not / There are not」です。

PositiveNegative
There is a cat.There is not a cat.
There are books.There are not books.

Questions: Is there / Are there

疑問文は文頭に置きます。

QuestionMeaning
Is there a pen?ペンはありますか。
Are there any apples?りんごはありますか。

✏️ 2. Short Example Sentences（短い例文）

  • There is a book on the desk.
    （机の上に本があります。）
  • There are two cats in the yard.
    （庭に猫が2匹います。）
  • There is a cup on the table.
    （テーブルの上にコップがあります。）
  • There are many students here.
    （ここにはたくさんの生徒がいます。）
  • There is a dog in the garden.
    （庭に犬がいます。）
  • There are apples in the box.
    （箱の中にりんごがあります。）
  • There is a park near my house.
    （家の近くに公園があります。）

📝 3. Longer Example Sentences（長い例文）

  • There is a small desk in my room, and there are books on it.
    （私の部屋には小さな机があり、その上に本があります。）
  • There are two windows, and there is a big bed near them.
    （窓が2つあり、その近くに大きなベッドがあります。）
  • There is a lamp on my desk, and there are pencils in a cup.
    （机の上にランプがあり、カップの中にえんぴつがあります。）
  • There are posters on the wall, and there is a chair under them.
    （壁にポスターがあり、その下にいすがあります。）
  • There is a small plant by the window, and there are flowers in it.
    （窓のそばに小さな植物があり、その中に花があります。）

📖 5. Short Reading Text（短い読解文）

My Room

My room is small, but I like it. There is a bed near the window. There are two chairs and a small desk. There is a lamp on the desk, and there are books under it. There is also a plant in the corner. My room is simple, but it feels nice.

私の部屋

私の部屋は小さいですが、気に入っています。窓の近くにベッドがあります。いすが2つと小さな机があります。机の上にランプがあり、その下に本があります。かどには植物もあります。シンプルですが、心地よい部屋です。

❓ 6. Reading Comprehension Questions（読解問題）

1. Is the room big or small?
（部屋は大きいですか、小さいですか。）
It is small.
2. Where is the bed?
（ベッドはどこにありますか。）
Near the window.
3. How many chairs are in the room?
（部屋にいすはいくつありますか。）
Two chairs.
4. What is on the desk?
（机の上に何がありますか。）
A lamp.
5. Where is the plant?
（植物はどこにありますか。）
In the corner.

🎯 Summary（まとめ）

  • There is = one thing. There are = many things.
    There is は単数、There are は複数です。
  • Use There is / There are to show something exists.
    存在を表すときに使います。
  • Negative forms: There is not / There are not.
    否定文は There is not / There are not。
  • Questions: Is there / Are there.
    疑問文は Is there / Are there。
  • This pattern helps describe rooms, places, and objects.
    部屋や場所、物の説明に便利です。
