E3: Likes & Preferences（好きなもの・きらいなもの）

📚 1. Grammar Explanation（文法の説明）

1. Using "like" and "don't like"

We use like to talk about things we enjoy, and don't like to talk about things we do not enjoy.

like は「好き」、don't like は「好きではない」を表します。

FormExample
I like …I like apples.
I don't like …I don't like carrots.
You like …You like music.
You don't like …You don't like soccer.

2. Talking about food and hobbies

We can use like with foods, sports, and hobbies.

食べ物、スポーツ、趣味にも like を使います。

  • I like pizza. – ピザが好きです。
  • I like reading. – 読書が好きです。
  • You like tennis. – あなたはテニスが好きです。

3. Giving simple reasons with "because"

We can explain our likes with because + reason.

because（なぜなら）を使って理由を言えます。

  • I like apples because they are sweet.
  • I don't like math because it is difficult.

🔤 New vocabulary (useful)

prefer — より好き
favorite — 一番好き

Try: "My favorite food is pizza." / "I prefer reading to watching TV."

✏️ 2. Short Example Sentences（短い例文）

  • I like apples.
    • りんごが好きです。
  • I don't like milk.
    • 牛乳は好きではありません。
  • You like music.
    • あなたは音楽が好きです。
  • I like swimming.
    • 私は水泳が好きです。
  • I don't like spiders.
    • 私はクモが好きではありません。
  • You like chocolate.
    • あなたはチョコレートが好きです。
  • I like games because they are fun.
    • ゲームが好きです。楽しいからです。

📝 3. Longer Example Sentences（長い例文）

  • I like cooking because I can make food for my family.
    • 料理が好きです。家族のために作れるからです。
  • You don't like coffee because it is too bitter.
    • あなたはコーヒーが好きではありません。苦すぎるからです。
  • I like reading books after school because it is relaxing.
    • 放課後に本を読むのが好きです。リラックスできるからです。
  • My sister likes dancing, and she practices every day.
    • 私の妹はダンスが好きで、毎日練習しています。
  • I don't like loud music because it hurts my ears.
    • 大きな音楽は好きではありません。耳が痛くなるからです。

🧩 4. Practice Exercises（練習問題）

Exercise A – Choose the correct option（正しい答えを選びましょう）

かっこ（　）の中から正しいほうをえらびましょう。

  1. I always smile when I eat apples. I ______ apples. ( like / don't like )
    I like apples.
  2. You always cheer when your team plays. You ______ soccer. ( like / don't like )
    You like soccer.
  3. I never finish my glass. I ______ milk. ( like / don't like )
    I don't like milk.
  4. I like pizza ______ it is delicious. ( because / and )
    because
  5. You don't like math ______ it is difficult. ( because / but )
    because

Exercise B – Complete the sentence（文を完成させましょう）

like / don't like / because を使って文を完成させましょう。

  1. I always want more chocolate. I ______ chocolate.
    I like chocolate.
  2. I don't like fish ______ it is smelly.
    because it is smelly.
  3. You always sing along to your favorite songs. You ______ music.
    You like music.
  4. I like running ______ it is fun.
    because it is fun.
  5. I run away when I see spiders. I ______ spiders.
    I don't like spiders.

Exercise C – Production (write 2–3 sentences)

次の問いに対して英語で2〜3文書いてみましょう。例は下の「Show Sample」。

  1. Write 2–3 sentences: What is your favorite food and why?
  2. Write 2–3 sentences: Do you prefer reading or watching TV? Explain.
Sample:
My favorite food is pizza because it is tasty and easy to share. I prefer reading to watching TV because reading helps me relax.

📖 5. Short Reading Text（短い読解文）

Likes and Dislikes

My name is Haru, and I am twelve years old. I like many things. I like apples because they are sweet. I also like drawing pictures after school. It is fun and relaxing. I don't like carrots because they taste strange to me. My friend Yuna likes soccer, but I don't like it because I am not good at running. We have different likes and dislikes, but we are still good friends.

好きなもの・きらいなもの

私の名前ははるで、12歳です。私は好きなものがたくさんあります。りんごが好きです。甘いからです。 放課後に絵を描くのも好きです。楽しくてリラックスできます。 にんじんは好きではありません。私には変な味がするからです。 友だちのゆなはサッカーが好きですが、私は走るのが得意ではないので好きではありません。 好きなもの・きらいなものはちがいますが、私たちは仲良しです。

❓ 6. Reading Comprehension Questions（読解問題）

  1. What food does Haru like?
    はるは何の食べ物が好きですか？
    Haru likes apples.
  2. Why doesn't Haru like carrots?
    はるはなぜにんじんが好きではありませんか？
    Because they taste strange to him.
  3. What hobby does Haru like?
    はるはどんな趣味が好きですか？
    Haru likes drawing pictures.
  4. Why doesn't Haru like soccer?
    はるはなぜサッカーが好きではありませんか？
    Because Haru is not good at running.
  5. Are Haru and Yuna still good friends?
    はるとゆなは今も仲良しですか？
    Yes, they are.
  6. What does Yuna like?
    ゆなは何が好きですか？
    Yuna likes soccer.

🎯 Summary（まとめ）

  • Use like and don't like to talk about preferences.
    • 好き・きらいを言うときに like / don't like を使います。
  • You can talk about food, hobbies, and activities.
    • 食べ物や趣味について話せます。
  • Use because to give simple reasons.
    • because で理由を言えます。
  • Everyone has different likes and dislikes.
    • 人によって好き・きらいはちがいます。
