CEFR: A1

Lesson 17: Some / Any

📚 1. Grammar Explanation（文法の説明）

We use some and any to talk about an unspecified amount.

「some」と「any」は、はっきりしない量について話すときに使います。

Use some in positive sentences. Use any in negative sentences and questions.

肯定文では some、否定文と疑問文では any を使います。

FormUseExample
somePositiveI have some apples.
anyNegative / QuestionsDo you have any apples?

Use some in questions when offering or requesting.

申し出や依頼の疑問文では some を使うことがあります。

✏️ 2. Short Example Sentences（短い例文）

  • I have some water.
    私は水を少し持っています。
  • She does not have any money.
    彼女はお金を持っていません。
  • Do you have any questions?
    何か質問はありますか。
  • I need some help.
    私は少し助けが必要です。
  • They do not have any food.
    彼らは食べ物を持っていません。
  • Can I have some water?
    水を少しいただけますか。
  • Would you like some tea?
    紅茶はいかがですか。

📝 3. Longer Example Sentences（長い例文）

  • I have some apples, but I do not have any oranges.
    私はりんごはいくつかありますが、オレンジはありません。
  • She needs some water because she does not have any drinks.
    彼女は飲み物がないので水が少し必要です。
  • Do you have any bread, or do you need some?
    パンはありますか、それとも少し必要ですか。
  • They do not have any time, so they need some help.
    彼らには時間がないので、少し助けが必要です。
  • Can I have some information about this event?
    このイベントについて少し情報をいただけますか。

📖 5. Short Reading Text（短い読解文）

The Empty Fridge

Our fridge is almost empty today. We do not have any milk, and there is no bread. My brother wants some juice, but we do not have any. My mother needs some vegetables for dinner, but there are none left. We need to go shopping soon, because we do not have any food for tonight.

空っぽの冷蔵庫

今日、冷蔵庫はほとんど空です。牛乳もなく、パンもありません。 弟はジュースがほしいですが、ありません。 母は夕食のために野菜が少し必要ですが、もう残っていません。 今夜の食べ物がないので、すぐに買い物に行く必要があります。

❓ 6. Reading Comprehension Questions（読解問題）

1) What is almost empty?
何がほとんど空ですか。

The fridge.

2) What does the brother want?
弟は何がほしいですか。

Some juice.

3) What does the mother need?
母は何を必要としていますか。

Some vegetables.

4) Why do they need to go shopping?
なぜ買い物に行く必要がありますか。

Because they do not have any food.

5) Do they have any bread?
パンはありますか。

No, they do not.

🎯 Summary（まとめ）

  • some is used in positive sentences.
    肯定文で some を使います。
  • any is used in negative sentences and questions.
    否定文と疑問文で any を使います。
  • some can appear in offers and requests.
    申し出や依頼では some を使うことがあります。
  • Use them for unspecified amounts.
    はっきりしない量について使います。
