CEFR: B1-B2

Lesson 10: Conditionals – Zero, First, Second, Third & Mixed

📚 1. Grammar Explanation（文法の説明）

1. Zero & First Conditional（0・1条件文）

Zero Conditional = general truths, rules. → いつも起こること・事実。 If + present → present

First Conditional = real future possibility. → 現実的な未来の可能性。 If + present → will + verb

TypeExample
ZeroIf you heat ice, it melts.
FirstIf it rains, we will stay home.

2. Second & Third Conditional（2・3条件文）

Second Conditional = unreal / imaginary present. → 現在の「もし〜なら」仮定。 If + past → would + verb

Third Conditional = unreal past. → 過去の「もし〜していたら」仮定。 If + had + past participle → would have + past participle

TypeExample
SecondIf I had more time, I would travel more.
ThirdIf she had studied, she would have passed.

3. Mixed Conditionals（ミックス条件文）

Used when the time in the “if” clause and the result clause are different. → if節と結果節の時間が異なるときに使います。

TypeExample
Past → PresentIf I had slept more, I would feel better now.
Present → PastIf I were more careful, I wouldn’t have made that mistake.

4. Common Japanese Learner Errors（日本人学習者のよくある間違い）

  • ❌ Using “will” after “if” → If it will rain… → 正しくは If it rains…
  • ❌ Mixing tenses incorrectly → If I study, I would pass. → 時制が合っていない。
  • ❌ Forgetting “had” in third conditional → If I studied, I would have passed. → 正しくは If I had studied…

✏️ 2. Short Example Sentences（短い例文）

  • If you mix red and blue, you get purple.
    赤と青を混ぜると紫になります。
  • If it rains tomorrow, we will cancel the trip.
    明日雨なら旅行を中止します。
  • If I had a car, I would drive to work.
    車があれば車で通勤するのに。
  • If she had left earlier, she would have caught the train.
    もっと早く出ていれば電車に間に合ったのに。
  • If I had slept more, I would feel better now.
    もっと寝ていれば今気分が良いのに。
  • If he were taller, he would play basketball.
    彼がもっと背が高ければバスケをするでしょう。
  • If you heat water to 100°C, it boils.
    水を100度まで加熱すると沸騰します。

📝 3. Longer Example Sentences（長い例文）

  • If you don’t follow the instructions carefully, the machine will not work properly.
    説明をよく読まないと、機械は正しく動きません。
  • If I had more free time, I would join a language exchange group to practice speaking.
    もっと時間があれば、会話練習のためにランゲージエクスチェンジに参加するのに。
  • If she had known about the meeting, she would have prepared the documents earlier.
    会議のことを知っていれば、もっと早く資料を準備していたでしょう。
  • If I hadn’t missed the bus, I would be at the office by now.
    バスに乗り遅れていなければ、今ごろオフィスにいるはずです。
  • If the weather improves tomorrow, we will go hiking in the mountains.
    明日天気が良くなれば、山にハイキングに行きます。

🧩 4. Practice Exercises（練習問題）

Exercise A – Choose the correct option（正しい答えを選びましょう）

1. If it rains, we ______ inside. (stay / will stay / stayed)
will stay
2. If I had more money, I ______ a new laptop. (buy / will buy / would buy)
would buy
3. If she had studied harder, she ______ the exam. (passes / would pass / would have passed)
would have passed
4. If you heat metal, it ______. (expands / expanded / will expand)
expands
5. If I hadn’t eaten so much, I ______ sick now. (wouldn’t feel / won’t feel / wouldn’t have felt)
wouldn’t feel

Exercise B – Complete the sentence（文を完成させましょう）

1. If I ______ (know) the answer, I would tell you.
knew
2. If she had left earlier, she ______ ______ (catch) the bus.
would have caught
3. If it rains tomorrow, we ______ (stay) home.
will stay
4. If I hadn’t been so tired, I ______ ______ (join) the event.
would have joined
5. If he were more organized, he ______ (make) fewer mistakes.
would make

📖 5. Short Reading Text（短い読解文）

Learning and Growing Through Workplace Challenges

Last year, I started a new job that required a lot of teamwork and communication. If I hadn’t taken an English course before joining the company, I would have struggled much more during the first few months. There were many situations where I needed to explain ideas clearly, and if I hadn’t practiced speaking, I wouldn’t have been able to participate confidently. My manager often said that if we worked together effectively, we would achieve better results, and he was right.

Now, I feel more comfortable expressing my thoughts, but I still make mistakes. If I had more time, I would study grammar more deeply, especially conditionals, because they appear everywhere in business communication. Sometimes I think that if I were more organized, I wouldn’t forget small tasks. Still, I try to improve little by little. If I continue learning consistently, I know I will become more confident and effective in my work.

職場での挑戦を通じた学びと成長

昨年、私はチームワークとコミュニケーションが多く必要な新しい仕事を始めました。 会社に入る前に英語コースを受けていなかったら、最初の数か月はもっと苦労していたと思います。 アイデアを明確に説明しなければならない場面が多く、話す練習をしていなければ、自信を持って参加できなかったでしょう。 マネージャーはよく「効果的に協力すれば、より良い結果が出る」と言っていて、その通りでした。

今では自分の考えを伝えることに慣れてきましたが、まだ間違えることもあります。 もっと時間があれば、文法、特に条件文をもっと深く勉強したいと思います。 ビジネスコミュニケーションでは条件文がよく使われるからです。 時々「もっと整理整頓ができていれば、小さなタスクを忘れないのに」と思うこともあります。 それでも少しずつ改善しようとしています。 継続して学び続ければ、もっと自信を持って効果的に働けるようになると感じています。

❓ 6. Reading Comprehension Questions（読解問題）

1. Why would the writer have struggled without the English course?
Because they needed to explain ideas clearly at work.
2. What did the manager say about teamwork?
He said that effective teamwork would lead to better results.
3. What would the writer study more deeply if they had more time?
Grammar, especially conditionals.
4. What mistake does the writer sometimes make?
They sometimes forget small tasks.
5. What does the writer believe will happen if they continue learning?
They will become more confident and effective at work.

🎯 7. Summary（まとめ）

  • 0 Conditional = facts.
    0条件文＝事実。
  • 1st Conditional = real future.
    1条件文＝現実の未来。
  • 2nd Conditional = unreal present.
    2条件文＝現在の仮定。
  • 3rd Conditional = unreal past.
    3条件文＝過去の仮定。
  • Mixed = different times.
    ミックス条件文＝時間が混ざる。
