CEFR: A2-B1

Lesson 8: Advice, Conditionals & Possession

📚 1. Grammar Explanation（文法の説明）

1. should / shouldn’t（助言）

We use should / shouldn’t + verb to give advice or say what we think is right. → 助言や「〜すべき／〜すべきでない」を言うときに使います。

  • You should sleep more.
  • You shouldn’t drink coffee at night.
  • I think you should talk to him.
  • I don’t think you should wait.

2. First Conditional（条件文）

We use if + present and will / won’t + verb to talk about possible future situations. → 未来の可能性について話すときに使います。

  • If you study, you’ll pass.
  • If it rains, we won’t go.
  • If you miss the bus, get a taxi.
  • You can call me if you’re late.

3. Possessive Pronouns（所有代名詞）

We use possessive pronouns to show who something belongs to. → 所有を表すときに使います。

✏️ 2. Short Example Sentences（短い例文）

  • You should eat more vegetables.
    • 野菜をもっと食べたほうがいいです。
  • You shouldn’t go to bed so late.
    • 遅く寝ないほうがいいです。
  • If it rains, we’ll stay home.
    • 雨が降ったら家にいます。
  • If you’re tired, take a break.
    • 疲れたら休んでください。
  • This book is mine.
    • この本は私のです。
  • Is that bag yours?
    • あのかばんはあなたのですか？
  • The keys are theirs, not mine.
    • その鍵は彼らのもので、私のではありません。

📝 3. Longer Example Sentences（長い例文）

  • You shouldn’t stay up too late if you have an exam tomorrow.
    • 明日試験があるなら、夜更かししないほうがいいです。
  • If we leave now, we’ll catch the train and arrive on time.
    • 今出発すれば、電車に間に合って時間通りに着きます。
  • I think you should talk to your manager before making a decision.
    • 決断する前に、上司と話したほうがいいと思います。
  • If you don’t bring your passport, you won’t be able to travel.
    • パスポートを持ってこなければ、旅行できません。
  • That phone isn’t mine. It’s hers.
    • その携帯は私のではありません。彼女のです。

🧩 4. Practice Exercises（練習問題）

Exercise A – Choose the correct option

1. You ______ go to bed earlier. (should / shouldn’t)
should
2. I don’t think you ______ eat that. (should / shouldn’t)
should
3. If you study, you ______ pass the test. (will / won’t)
will
4. If it rains, we ______ go to the park. (will / won’t)
won’t
5. That pen is ______. (mine / my)

Exercise B – Complete the sentence

1. You ______ to bed earlier. You look exhausted. (go)
should go
2. If it ______ tomorrow, we won't go to the park. (rain)
rains
3. That jacket isn't mine. It's ______. (she)
hers
4. If you ______ now, you'll catch the last train. (leave)
leave
5. I don't think you ______ anyone about this. It's private. (tell)
should tell

📖 5. Short Reading Text（短い読解文）

Emma’s Helpful Advice

My friend Emma loves giving advice. She thinks I should eat healthier and sleep more. Last week, she told me I shouldn’t drink coffee at night. I listened to her, and now I sleep better. She also said that if I exercise regularly, I’ll feel more energetic. Emma is very active, and she goes running every morning. If it rains, she’ll still go—she just takes an umbrella! I admire her discipline, and I think I should try to follow her example.

エマの役に立つアドバイス

友達のエマはアドバイスをするのが大好きです。彼女は、私がもっと健康的に食べて、もっと寝るべきだと言います。先週、夜にコーヒーを飲むべきではないと言われました。言われた通りにしたら、よく眠れるようになりました。また、定期的に運動すればもっと元気になると言いました。エマはとても活動的で、毎朝ランニングをします。雨が降っても走ります。傘を持っていくだけです。彼女の習慣を尊敬していて、私も見習うべきだと思っています。

❓ 6. Reading Comprehension Questions（読解問題）

1. What does Emma think the writer should do?

エマは筆者が何をすべきだと言っていますか？

Eat healthier and sleep more.
2. What happened when the writer stopped drinking coffee at night?

夜にコーヒーを飲むのをやめたらどうなりましたか？

They slept better.
3. What will happen if the writer exercises regularly?

定期的に運動すればどうなりますか？

They will feel more energetic.
4. What does Emma do every morning?

エマは毎朝何をしますか？

She goes running.
5. Does Emma run even if it rains?

雨が降ってもエマは走りますか？

Yes, she takes an umbrella and still goes.

🎯 7. Summary（まとめ）

  • should / shouldn’t: advice
    • 助言をするときに使う。
  • First conditional: if + present → will / won’t + verb
    • 未来の可能性について話す。
  • Possessive pronouns: mine, yours, his, hers, ours, theirs
    • 所有を表す代名詞。
  • Don’t use a noun after possessive pronouns.
    • 所有代名詞の後に名詞は置かない。
